# Stoop Sale Event in Brooklyn

This project is a web page promoting a stoop sale event in Brooklyn. The page includes a countdown timer, an interactive map, social media sharing buttons, and an integrated Spotify player.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Leaflet.js for the interactive map
- Google Calendar API for adding reminders
- Spotify Embed for the music player
- Google Fonts

## Key Features
- **Responsive Design:** The page adapts to different screen sizes.
- **Countdown Timer:** Displays the time remaining until the event.
- **Interactive Map:** Uses Leaflet.js to show the event location.
- **Reminder Button:** Allows users to add the event to their Google Calendar.
- **Social Media Sharing:** Includes buttons to share on LinkedIn and other platforms.
- **Spotify Player:** A toggle button that shows/hides an embedded Spotify player.
- **Custom Styling:** Uses custom fonts and CSS styles to create a unique appearance.

## Project Structure
- `index.html`: Main structure of the page.
- `style.css`: CSS styles for the page design.
- `script.js`: JavaScript functionalities, including the countdown timer and map integration.
- `image.png`: Logo or main image of the event.

## How to Use
1. Clone the repository or download the files.
2. Open `index.html` in a modern web browser.
3. For development, you can edit the HTML, CSS, and JavaScript files as needed.

## Customization
- Modify the event date in `script.js`.
- Update the map coordinates in the `initMap()` function.
- Change styles in `style.css` to adjust colors and design.
- Replace `image.png` with your own event logo or image.


## Vercel Deployment
https://codedex-hackathon-track-2.vercel.app/

## Additional Notes
- The sharing functionality uses the Web Share API, which may not be compatible with all browsers.
- Ensure you have an internet connection to load Google Fonts, the Leaflet map, and the Spotify player.

### 3. Write a description of your project
Our project, "Stoop Sale Invitation for Brooklyn", is an interactive web invitation designed for an upcoming Stoop Sale event in Brooklyn. The aim was to create an engaging and immersive experience that captures the essence of community spirit and excitement surrounding the event.

**Key Features:**

- **Interactive Elements:** The invitation incorporates interactive features such as music playback, animations, and vibrant visuals to convey the lively atmosphere of the Stoop Sale.
  
- **Countdown Timer:** A prominently displayed countdown timer builds anticipation and informs visitors about the event's date and time, ensuring they don't miss out.

- **Location Map:** Utilizing Leaflet.js, we integrated an interactive map that allows attendees to visualize the exact location of the Stoop Sale. This feature enhances convenience and accessibility for potential visitors.

- **External Sharing:** Social media sharing buttons enable users to easily spread the word about the event, fostering community engagement and expanding reach beyond immediate networks.

- **Positive Vibes:** Throughout the design process, emphasis was placed on creating a visually appealing and welcoming atmosphere that reflects the event's vibrant and inclusive nature.

**Technologies Used:**

- **HTML5, CSS3, JavaScript:** These core technologies were used to build the foundation of the interactive invitation, ensuring compatibility across various devices and browsers.
  
- **Leaflet.js:** Integrated for the interactive map functionality, providing a user-friendly interface for location visualization.

- **Google Calendar API:** Implemented to allow users to add event reminders directly to their calendars, enhancing user engagement and participation.

- **Spotify Embed:** Incorporated to provide a seamless music listening experience, adding to the festive ambiance of the event.

**Challenges Faced:**

- **Integration Complexity:** Integrating multiple APIs and ensuring seamless functionality posed initial challenges. However, thorough testing and iterative improvements ensured a smooth user experience.

- **Responsive Design:** Ensuring optimal performance across different screen sizes and devices required meticulous attention to detail in CSS styling and layout adjustments.

- **User Experience:** Balancing creativity with usability was key to creating an invitation that not only looks appealing but also functions intuitively for all users.

This project was a testament to our commitment to delivering an interactive and memorable experience for the Stoop Sale attendees, encapsulating the spirit of community and celebration. By leveraging modern web technologies and creative design elements, we aimed to exceed expectations and create a lasting impression.
